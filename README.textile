h2. Introduction

A simple parser for converting UK Bill pdf documents from http://services.parliament.uk/bills into comparable plain text files

Requires the pdftotext utility, part of Xpdf http://www.foolabs.com/xpdf/download.html


h2. Example Usage

bc. ruby ./parse.rb 'Digital Economy Bill [HL]' 2010089.pdf DigitalEconomyBill-089-0910.txt
